If you hit a blocker while running a Generator or Host, or while working through the docs, there are two places to go: the [Shinzo Discord](https://discord.shinzo.network) for onboarding and technical support, and GitHub Issues for reporting bugs and requesting features.

## Onboarding and technical support

Join the [Shinzo Discord](https://discord.shinzo.network) for help getting set up or troubleshooting a running node. It's the fastest channel for onboarding questions.

## Report an issue

Open a GitHub issue in the repo that matches what you ran into:

- [shinzo-generator-client](https://github.com/shinzonetwork/shinzo-generator-client/issues) for technical issues with the Generator client.
- [shinzo-host-client](https://github.com/shinzonetwork/shinzo-host-client/issues) for technical issues with the Host client.
- [docs](https://github.com/shinzonetwork/docs/issues) for bugs, broken links, missing pages, or feature requests about these docs.

## What to include in a report

Include the following where it applies:

1. What you did (the exact commands or steps, not a summary)
1. What you expected to happen.
1. What happened instead (paste the full error message or log output, not just the last line)
1. Versions. The client version from the `/health` endpoint or the binary's `--version` flag, the source chain, and your OS.
1. For registration problems, the transaction hash and your DID. You can find your DID in the Technical Registry after registering.
